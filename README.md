## Platform Map: 

a. User (Buyer / Seller): 

Is the customer or client of shops.

Consists of: 

- First Name
- Last Name 
- Phone number 
- Email 
- Password
- User Role (Buyer or Seller) 
- Company Name


d. Products: 

Are the tools / equipment to be sold or rented by the shop 

- Consists of: 
  - Image 
  - Name
  - Price
  - Description 
  - Quantity 
  - Details 

e. Store

Is the shop or company that will offer its products to the clients
